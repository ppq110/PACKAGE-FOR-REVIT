BIMLab DynLock - Member Package

Muc tieu:
- Cai add-in vao Revit 2024/2025/2026
- Dang nhap
- Load plugin .dynx do leader gui
- Chay cong cu trong Revit

Thu muc chinh:
- `C:\ProgramData\BIMLab\DynLock\`
- `C:\ProgramData\BIMLab\DynLock\Scripts\`

Luong su dung:
1. Chay `BIMLab Player.exe`
2. Chon Revit version can cai
3. Bam `Cai add-in`
4. Mo Revit
5. Bam `Login`
6. Bam `Load` va chon file `.dynx`
7. Xem preview: Panel, Plugin, Icon
8. Bam nut plugin vua duoc nap trong ribbon de chay

Neu can chon CAD:
- Form inputs se hien nut `Chon...`
- Chon CAD trong Revit, bam Finish
- Sau do quay lai form de tiep tuc

Neu co loi:
- Kiem tra da cai dung version Revit chua
- Kiem tra file `.dynx` co duoc leader gui dung khong
- Kiem tra package Dynamo co ton tai trong `%AppData%\Dynamo\Dynamo Revit\<version>\packages`
